## Question Backend

Backend Golang Server for Question project.
Question is a web page for the event attendee can ask the speaker the anonymous question.

## Language

Golang

## How to run

Setup development evironment
```
make init
```

Run project
```
make run
```

## Features

- [ ] Make a new topic
- [ ] Close the topic
- [ ] Submit question
- [ ] Mark done question
- [ ] Upvote question
